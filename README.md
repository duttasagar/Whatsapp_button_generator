# Whatsapp_button_generator

Application Description:
This is a modern, responsive, and interactive web application built using React. It features dynamic routing with React Router DOM, intuitive UI components with React Icons, and is styled with Tailwind CSS for rapid and consistent design. For enhanced visuals and animations, OGL (WebGL framework) is used to create smooth 3D graphics and interactive effects. The project is bundled and served using Vite, ensuring fast development and optimized builds.

Tech Stack:

Frontend: React, React DOM, React Router DOM, React Icons
Styling: Tailwind CSS, @tailwindcss/vite
Animation / Graphics: OGL (WebGL framework)
Build Tool / Bundler: Vite
